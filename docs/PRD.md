# PRD

## Product
Portifolio premium de Bernardo Kawano com foco em AI Engineering e automacoes de operacoes.

## Goals
1. Impressionar recrutadores tecnicos
2. Gerar interesse de consultoria
3. Exibir projetos como estudos de caso
4. Facilitar contato com foco em WhatsApp

## Required Pages
- Home
- Projects
- About
- Contact

## Functional Requirements
- Website bilingue (`pt`, `en`)
- Deteccao automatica de idioma inicial
- Troca manual de idioma com persistencia
- Dark/light mode com deteccao automatica
- Troca manual de tema com persistencia
- Estrutura de dados editavel para projetos e links

## Non-functional Requirements
- Responsivo (mobile primeiro)
- Boa acessibilidade
- SEO basico por pagina
- Performance visual e animacoes discretas
